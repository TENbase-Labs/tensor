# @tenbaselabs/tensor-themes

Theme packages for the Tensor design system (PlusOne, HooverSheet, etc.).

## Installation

```bash
npm install @tenbaselabs/tensor-themes @tenbaselabs/tensor
```

## Usage

```typescript
import { plusone } from '@tenbaselabs/tensor-themes/plusone';
import { hooversheet } from '@tenbaselabs/tensor-themes/hooversheet';

// Apply theme
document.documentElement.classList.add(plusone);
```

## Available Themes

- **PlusOne**: Primary brand theme
- **HooverSheet**: Secondary brand theme

## Features

- CSS-based theming
- Runtime theme switching
- Custom property overrides
- Brand-specific tokens
