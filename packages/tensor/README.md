# @tenbaselabs/tensor

Core design system tokens and utilities for TENbase Labs products.

## Installation

```bash
npm install @tenbaselabs/tensor
```

## Usage

```typescript
import { tokens } from '@tenbaselabs/tensor';
```

## Features

- Design tokens (colors, typography, spacing)
- CSS custom properties
- Theme utilities
- Multi-brand support
