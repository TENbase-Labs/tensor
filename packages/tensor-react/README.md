# @tenbaselabs/tensor-react

React components for the Tensor design system.

## Installation

```bash
npm install @tenbaselabs/tensor-react @tenbaselabs/tensor react react-dom
```

## Usage

```tsx
import { Button, Card } from '@tenbaselabs/tensor-react';

function App() {
  return (
    <Card>
      <Button variant="primary">Click me</Button>
    </Card>
  );
}
```

## Features

- Fully typed React components
- Theme-aware styling
- Accessibility built-in
- Multi-brand support
